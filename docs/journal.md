# Project Journal

## Step 1 – 24.04

### Planned System and Goal

The planned system is an AI-assisted study assistant implemented in Python. The goal of the system is to help users solve simple study-related questions by using external tools during execution.

The user enters a question or task. The system analyzes the request, chooses a suitable tool, uses the tool, and returns a structured answer.

### AI or Agent-Based Approach

The system uses a simple agent-based workflow.

The agent receives the user input, decides which tool is needed, runs the selected tool, and prepares the final response.

For example, if the user enters a mathematical expression, the calculator tool will be used. If the user provides a file path, the file reader tool will be used.
