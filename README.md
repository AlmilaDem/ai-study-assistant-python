# AI Study Assistant Python System

## Project Description

This project is an AI-assisted Python study assistant that uses a simple agent-based workflow. The system receives user input, analyzes the request, selects a suitable tool, and returns a structured result.

The goal of the project is to demonstrate how an AI-style assistant can use external tools during execution to solve practical problems.

## Planned Features

- User input handling
- Agent-based decision logic
- Calculator tool for mathematical expressions
- File reader tool for reading text files
- Input validation
- Structured output formatting
- Unit testing
- Deployment preparation
